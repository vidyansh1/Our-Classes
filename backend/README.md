# OwnClasses Backend

This is a MERN-ready Express and MongoDB backend scaffold for OwnClasses.

## Included

- User model with student, teacher, and admin roles
- Authentication routes for signup and login
- Course APIs for creating and listing courses
- Live class APIs for scheduling and listing classes
- Environment configuration for MongoDB, JWT, and meeting providers

## Quick start

1. Copy `.env.example` to `.env`
2. Add your MongoDB connection string
3. Install packages with `npm install`
4. Run with `npm run dev`
